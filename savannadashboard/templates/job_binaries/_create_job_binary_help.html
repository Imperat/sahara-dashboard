Create a Job Binary with a specified name.

Select the type of your Data Source.

  • Savanna internal database
  • Swift internal (to openstack)

For Savanna internal job binaries, you may choose from the following:

  • Choose an existing file
  • Upload a new file
  • Create a script to be uploaded dynamically

For Swift internal job binaries, you must:

  • Enter the URL for the file
  • Enter the username and password required to access that file

You may also enter an optional description for your Job Binary.
